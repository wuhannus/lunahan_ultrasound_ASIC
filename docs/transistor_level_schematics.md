  M1:  Input NMOS, W=200Вөm/0.15Вөm, 4 fingers (for low Rg, optimal NFmin)
  MCAS: Cascode NMOS, W=200Вөm/0.18Вөm (boosts gain, improves reverse isolation)
  LLOAD: Inductive load, 1mH (resonates with output C at 40 kHz)
  RLOAD: 8kО© parallel damping resistor
  LS: Source degeneration inductor, 100ВөH (creates real 50О© input impedance)
  Lg: Gate inductor, 330ВөH (off-chip, resonates with Cgs at 40 kHz)
  M2: Gain NMOS, W=100Вөm/0.15Вөm
  MLOAD: PMOS active load, W=40Вөm/0.5Вөm
  M3: Source follower NMOS, W=80Вөm/0.15Вөm
  MSINK: Current sink NMOS, W=20Вөm/0.5Вөm

Bias: PTAT constant-gm reference вҶ’ mirrored to all stages
  I(M1) вүҲ 250ВөA  (set for NFmin)
  I(M2) вүҲ 200ВөA
  I(M3) вүҲ 100ВөA
